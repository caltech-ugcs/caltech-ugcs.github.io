Map and Filter

Often, the response of a REST API can be processed with a combination of two functions, map and reduce.

The map((x, i) => {...}) method returns a list constructed by applying a function to each element of a list.

let arr = [1, 2, 3]
let arr2 = arr.map((x, i) => x * x) // [1, 4, 9]

The filter((x, i) => {...}) method returns the sublist of elements for which the given function evaluates to true.

let arr = [1, 2, 3, 4, 5]
let arr2 = arr.filter((x, i) => x % 2 == 1) // [1, 3, 5]
