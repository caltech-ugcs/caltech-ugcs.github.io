Access Class Schedule

The built in fetch function can be used to make HTTP requests to a REST API. The documentation for the fetch function can be found here. The basic usage of the fetch is as follows.

// fetch the data at the given url.
let response = await fetch(url);
// parse the json payload.
let payload = await response.json();

Attempt to modify the 'fetchClasses' function to make a HTTP GET request for the course catalog data.
